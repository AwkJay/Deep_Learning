# Deep Learning Summer School

This repository contains my work from a deep learning summer school conducted at IIITDM Jabalpur. The course was structured over multiple weeks and covered both theory and hands-on sessions across different areas of deep learning.

I’ve organized the repo week-wise, roughly following how the course progressed.

---

## Structure

The content is divided based on the weekly flow of the program:

* **Week 1**
  Introduction to AI/ML/DL, neural networks, perceptron, optimization, and basics of TensorFlow/PyTorch

* **Week 2**
  CNNs, transfer learning, RNNs, transformers, autoencoders, and introduction to LLMs

* **Week 3**
  GANs, diffusion models, explainable AI, and reinforcement learning

* **Week 4**
  Applications like medical imaging, NLP, remote sensing, and final project discussions

---

## What’s in the repo

Each folder contains a mix of material from that week:

* lecture slides and PDFs
* practice questions
* quiz files (sometimes split into questions and answers)
* notebooks for hands-on work
* some solution files

This is more of a working repo than a clean final submission, so things may be a bit rough in places.

---

## How the course was structured

Each day usually had:

* a theory session (concepts and background)
* followed by a practical session (implementation or hands-on work)

Topics ranged from basic neural networks to more recent areas like diffusion models and LLMs, along with some exposure to real-world applications.

---

## Tools used

Most of the work here is in Python using:

* PyTorch and TensorFlow
* NumPy, Pandas
* scikit-learn
* matplotlib
* some Hugging Face libraries (for transformers/diffusion models)
* LIME for explainability
