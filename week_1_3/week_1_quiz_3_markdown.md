## Questions

1. Which of the following is a commonly used loss function for binary classification?
- a) Categorical Cross-Entropy
- b) Mean Absolute Error
- c) Binary Cross-Entropy
- d) Mean Squared Error

2. What does gradient descent aim to minimize during training?
- a) Accuracy
- b) Learning rate
- c) Loss function
- d) Number of neurons

3. In neural networks, what does the learning rate control?
- a) Step size in weight updates
- b) Model complexity
- c) Number of layers
- d) Number of epochs

4. What does a vanishing gradient problem usually affect?
- a) Batch size
- b) Model architecture
- c) GPU performance
- d) Deep neural networks with sigmoid/tanh

5. What is the purpose of using the ReLU activation function?
- a) To introduce non-linearity
- b) To increase learning rate
- c) To normalize the input
- d) To compute probability

6. If the learning rate is too high, what could happen during training?
- a) Model will train slowly
- b) Weights will become constant
- c) Loss may oscillate or diverge
- d) Model will underfit

7. What does batch size determine in neural network training?
- a) Number of hidden layers
- b) Number of models trained
- c) Number of samples used to compute gradient
- d) Number of neurons per layer

8. Which term best describes “adjusting learning rate, batch size, and optimizer”?
- a) Data augmentation
- b) Hyperparameter tuning
- c) Model pruning
- d) Model validation

9. Why is Binary Cross Entropy used in binary classification tasks?
- a) It models probabilities of two classes
- b) It performs weight regularization
- c) It minimizes learning rate
- d) It penalizes correct predictions

10. Which of the following plots is most useful to track convergence during training?
- a) Histogram of weights
- b) Scatter plot of input features
- c) Bar chart of accuracy
- d) Loss vs Epoch line plot

11. In SGD (Stochastic Gradient Descent), the update is performed:
- a) After every batch or sample
- b) Only once before training
- c) After processing the full dataset
- d) Once after convergence

12. What is the effect of a large batch size on training?
- a) Less noise, slower updates
- b) Slower computation, less noise
- c) More noise, faster convergence
- d) More noise, slower convergence

13. What is a drawback of using only Mean Squared Error in classification?
- a) It computes probabilities incorrectly
- b) It doesn’t account for softmax
- c) It penalizes large errors less
- d) It’s not differentiable

14. What is the purpose of weight initialization?
- a) Prevents vanishing gradients
- b) Selects loss function
- c) Fixes batch size
- d) Avoids symmetry in updates

15. During training, if loss is decreasing but accuracy is not improving, what could be the issue?
- a) More epochs needed
- b) Class imbalance or wrong threshold
- c) Too low learning rate
- d) Overfitting
