## Questions

1. Which feature makes RNNs different from traditional feedforward neural networks?
- a) Ability to work without activation functions
- b) No hidden layers
- c) Shared weights across time steps

- b) Low computational power
- c) Lack of sufficient training data
- d) Vanishing gradient problem

3. In an RNN, what is typically passed from one time step to the next?

- b) Hidden state
- c) Output label
- d) Loss value

4. Which of the following is not a gate used in an LSTM cell?

- b) Output gate
- c) Input gate
- d) Memory gate

5. Why are gates important in LSTM and GRU architectures?

- b) They reduce training time

- b) GRUs have separate memory cells

- b) No output at all
- c) A fixed image representation
- d) An output at each word position

8. Which application is best suited for a many-to-one RNN architecture?

- b) Part-of-speech tagging

9. What is the main benefit of using bidirectional RNNs in NLP tasks?

- b) They reduce model size
- c) They convert text to images
