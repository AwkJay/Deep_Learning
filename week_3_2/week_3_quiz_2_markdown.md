# Week 3 Quiz 2: Diffusion Models

## Questions

1. What is the core idea behind a diffusion model?
- a) Compressing images using a neural network
- b) Generating labels for input data
- c) Learning to reverse a gradual noise-adding process
- d) Translating images to text

2. In a Denoising Diffusion Probabilistic Model (DDPM), what happens during the forward process?
- a) Noise is removed from the image
- b) Gaussian noise is added step by step
- c) The model generates new captions
- d) The image is upscaled to higher resolution

3. Which distribution is typically used to add noise in the forward process of DDPM?
- a) Uniform
- b) Poisson
- c) Binomial
- d) Gaussian (Normal)

4. What does the reverse process in a diffusion model aim to do?
- a) Compress image features
- b) Remove noise and reconstruct original data
- c) Predict image labels
- d) Create segmentation masks

5. Which loss function is commonly used to train DDPMs?
- a) Categorical Cross Entropy
- b) Hinge Loss
- c) Mean Squared Error
- d) Binary Cross Entropy

6. What is the role of the beta schedule in a DDPM?
- a) It controls the learning rate decay
- b) It decides the optimizer to use
- c) It adjusts the resolution of the images
- d) It defines the variance of noise added at each timestep

7. In Stable Diffusion, how is text information incorporated into the image generation process?
- a) It is added as a watermark
- b) It is not used
- c) It is encoded into image pixels
- d) It is embedded using CLIP text encoder

8. What kind of model is typically used for the denoising network in diffusion models?
- a) LSTM
- b) U-Net
- c) ResNet
- d) Decision Tree

9. Why are diffusion models considered more stable than GANs during training?
- a) They use adversarial loss
- b) They avoid mode collapse and adversarial game
- c) They train faster
- d) They use more layers

10. What is one major advantage of using pretrained Stable Diffusion models?
- a) They are faster than CNNs for classification
- b) They only generate cartoons
- c) They require zero memory
- d) They allow high-quality image generation from text prompts
