## Questions

1. In reinforcement learning, what is the agent trying to learn?
- a) The best actions to maximize reward over time
- b) How to classify images
- c) A sequence of random actions
- d) A labeled output for each input

2. What is the "environment" in RL?
- a) The space in which the agent acts and receives rewards
- b) A database
- c) A loss function
- d) A program that supervises learning

3. What is a "reward" in RL?
- a) Feedback given to the agent for an action
- b) A type of model
- c) A correct label
- d) A loss value

4. In Q-learning, what does the agent learn?
- a) A Q-table that tells the value of taking an action in a state
- b) A list of best states
- c) A mapping from actions to states
- d) A label for each input

5. In RL, an "episode" refers to:
- a) A full run from start to goal
- b) A batch of training data
- c) A random action
- d) A type of optimizer

6. What is Multi-Agent RL?
- a) Multiple agents learning in the same environment
- b) A type of data augmentation
- c) One agent using many models
- d) A supervised model for multi-class output

7. In Multi-Agent RL, agents can:
- a) Work together, compete, or both
- b) Only work one at a time
- c) Only compete with each other
- d) Share the same model weights always

8. In Multi-Agent RL, a key challenge is:
- a) The environment becomes non-stationary as other agents learn
- b) Computing gradients efficiently
- c) Having labeled data
- d) Lack of a reward signal

9. What does the "experience replay" mechanism in Deep Q-Learning do?
- a) Reuses past experiences to improve learning
- b) Randomly chooses new agents
- c) Deletes old states
- d) Stores final scores

10. Which of the following describes the exploration vs exploitation dilemma?
- a) Balancing between trying new actions and using known good ones
- b) Switching between models
- c) Choosing between learning and forgetting
- d) Deciding whether to stop learning
